- .... ومن الناس من يقول : لا اجماع الا لعترة الرسول لأنهم المخصوصون بقرابة رسول الله (ص) وأسباب العز ، قال : (ع) : إني تارك فيكم الثقلين ، كتاب الله وعترتي إن تمسكتم بهما لم تضلوا بعدي ، وقال تعالى : { إِنَّمَا يُرِيدُ اللهُ لِيُذْهِبَ عَنكُمُ الرِّجْسَ أَهْلَ الْبَيْتِ وَيُطَهِّرَكُمْ تَطْهِيرًا ( الأحزاب : 33 ) } ولكنا نقول : أنواع الكرامة لأهل البيت ، متفق عليه.

 

 

الشبراوي - الاتحاف بحب الأشراف

الجزء : ( 1 ) - رقم الصفحة : ( 5 )

 

[ النص طويل لذا استقطع منه موضع الشاهد ]

 

- .... واكبارا لمقامهم لقوله تعالى : { إِنَّمَا يُرِيدُ اللهُ لِيُذْهِبَ عَنكُمُ الرِّجْسَ أَهْلَ الْبَيْتِ وَيُطَهِّرَكُمْ تَطْهِيرًا ( الأحزاب : 33 ) } روى الترمذي ، عن عمر بن أبي سلمة ربيب النبي (ص) ، قال : لما نزلت هذه الآية : { إِنَّمَا يُرِيدُ اللهُ لِيُذْهِبَ عَنكُمُ الرِّجْسَ أَهْلَ الْبَيْتِ وَيُطَهِّرَكُمْ تَطْهِيرًا ( الأحزاب : 33 ) } في بيت أم سلمة (ر) دعا فاطمة وحسنا وحسينا وجللهم بكساء وعلي خلف ظهره ، ثم قال : اللهم هؤلاء أهل بيتي أذهب عنهم الرجس وطهرهم تطهيرا.

 

- .... روى الترمذي ، عن عمر بن أبي سلمة ربيب النبي (ص) ، قال : لما نزلت هذه الآية : { إِنَّمَا يُرِيدُ اللهُ لِيُذْهِبَ عَنكُمُ الرِّجْسَ أَهْلَ الْبَيْتِ وَيُطَهِّرَكُمْ تَطْهِيرًا ( الأحزاب : 33 ) } في بيت أم سلمة (ر) دعا فاطمة وحسنا وحسينا وجللهم بكساء وعلي خلف ظهره ، ثم قال : اللهم هؤلاء أهل بيتي أذهب عنهم الرجس وطهرهم تطهيرا.
