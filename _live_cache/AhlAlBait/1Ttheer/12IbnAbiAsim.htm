العودة لقائمة الرواة

 

{ إِنَّمَا يُرِيدُ اللهُ لِيُذْهِبَ عَنكُمُ الرِّجْسَ أَهْلَ الْبَيْتِ وَيُطَهِّرَكُمْ تَطْهِيرًا ( الأحزاب : 33 ) }

 

 عدد الروايات : ( 2 )

 

ابن أبي عاصم - الآحاد والمثاني - النساء

فاطمة ابنة رسول الله (ص) ، أم الحسن والحسين ، زوجة علي بن أبي طالب (ر)

توفيت بعد رسول الله (ص) بستة أشهر وهي ابنة ، ثمان وعشرين سنة

وكان مولدها وقريش تبني الكعبة وغسلها علي (ر) ، ودفنها ليلا ، وصلى

 الجزء : ( 5 ) - رقم الصفحة : ( 360 )

 

[ النص طويل لذا استقطع منه موضع الشاهد ]

 

2953 - حدثنا : هدبة بن خالد ، نا : حماد بن سلمة ، عن علي بن زيد ، عن أنس (ر) : أن النبي (ص) كان يمر ببيت فاطمة (ر) ستة أشهر إذا خرج إلى صلاة الفجر ، فيقول : يا أهل البيت : { إِنَّمَا يُرِيدُ اللهُ لِيُذْهِبَ عَنكُمُ الرِّجْسَ أَهْلَ الْبَيْتِ وَيُطَهِّرَكُمْ تَطْهِيرًا ( الأحزاب : 33 ) }.

 

 

ابن أبي عاصم - كتاب السنة - باب : ما ذكر في فضل علي (ر)

الجزء : ( 2 ) - رقم الصفحة : ( 603 )

 

[ النص طويل لذا استقطع منه موضع الشاهد ]
