- .... وزيادة على ذلك ، قال عز وجل : { فَمَنْ حَاجَّكَ فِيهِ مِن بَعْدِ مَا جَاءَكَ مِنَ الْعِلْمِ فَقُلْ تَعَالَوْا نَدْعُ أَبْنَاءَنَا وَأَبْنَاءَكُمْ وَنِسَاءَنَا وَنِسَاءَكُمْ وَأَنفُسَنَا وَأَنفُسَكُمْ ثُمَّ نَبْتَهِلْ فَنَجْعَل لَّعْنَتَ اللَّهِ عَلَى الْكَاذِبِينَ ( آل عمران : 61 ) } ولم يدع رسول الله (ص) عند مباهلة النصارى غير علي وفاطمة والحسن والحسين عليهم السلام.

 الجزء : ( 1 ) - رقم الصفحة : ( 171 )

 

[ النص طويل لذا استقطع منه موضع الشاهد ]

 

- .... ولما نزلت هذه الآية : { فَقُلْ تَعَالَوْا نَدْعُ أَبْنَاءَنَا وَأَبْنَاءَكُمْ وَنِسَاءَنَا وَنِسَاءَكُمْ وَأَنفُسَنَا وَأَنفُسَنَا وَأَنفُسَكُمْ ( آل عمران : 61 ) } دعا رسول الله (ص) عليا وفاطمة والحسن والحسين عليهم السلام ، وقال : اللهم هؤلاء أهلي ، خرجه مسلم ، والترمذي.
