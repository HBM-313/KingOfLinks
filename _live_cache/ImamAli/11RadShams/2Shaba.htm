العودة لصفحة البداية

العودة لفهرس المواضيع

 

( الصحابة الذين رووا حديث رد الشمس )

 

 عدد الصحابة الرواة : ( 1 )

 

الشيخ المحمودي - رسائل في حديث رد الشمس

 رقم الصفحة : ( 15 )

 

- وأما رواة الحديث من أصحاب رسول الله (ص) فالذين وجدنا حديثهم تسعة :

 

الأول : الامام أمير المؤمنين (ع ) ، وحديثه رواه الحافظ الحسكاني وأبو الحسن شاذان الفضلي ، ويجد الباحث الحديث في آخر الأحاديث التي علقناها على الحديث ( ( 815 ) من ترجمة أمير المؤمنين (ص) من تاريخ دمشق - الجزء ( 2 ) - رقم الصفحة ( 303 ) - الطبعة الثانية ).

 

الثاني : الامام الحسين (ع ) ، وحديثه تحت ( الرقم ( 158 ) من كتاب الذرية الطاهرة الورق ( 28 / ب ) ) وقد مر آنفا.

 

الثالث : جابر ابن عبد الله الأنصاري ، وحديثه في أواخر الفصل ( ( 19 ) من مناقب الخوارزمي - الصفحة ( 236 ) ) ، ورواه أيضا شاذان الفضلي في الحديث الأخير من رسالته في رد الشمس ، والطبراني في الأوسط كما في تعليقي على ( تاريخ دمشق - الجزء ( 2 ) - رقم الصفحة ( 301 ) ).

 

الرابع : أبو رافع مولى رسول الله (ص) ، وحديثه تحت الرقم ( ( 141 ) من مناقب ابن المغازلي - رقم الصفحة ( 98 ) ).

 

الخامس : أبو سعيد الخدري رفع الله مقامه ، وحديثه في رسالة رد الشمس للحافظ الحسكاني.

 

السادس : أبو هريرة ، وحديثه في رسالة أبي الحسن شاذان الفضلي ورسالة الحافظ الحسكاني.

 

السابع : أنس بن مالك كما رواه عنه محمد بن سليمان المتوفى عام ( 322 ) في الحديث ( 1020 ) من مناقب أمير المؤمنين (ص) الجزء ( 2 ) - رقم الصفحة ( 516 ) - الطبعة الأولى.

 

الثامن : عبد الله بن العباس ، وحديثه في المنقبة ( 75 ) من مائة منقبة لابن شاذان - رقم الصفحة ( 143 ) - وفي الحديث : ( 72 ) من الفصل ( 19 ) من مناقب الخوارزمي - رقم الصفحة ( 236 ).

 

التاسع : الصحابية أسماء بنت عميس رفع الله مقامها ، ويصح عد حديثها متواترا بالمعنى لكثرة أسانيدها ومصادرها.

 

- وجميع من ذكرناهم من الصحابة هاهنا ذكرنا حديثه حرفيا من وجوه أكثر مما أشرنا إليه هاهنا في تعليق الحديث ( 814 / 816 ) من ترجمة أمير المؤمنين (ص) من تاريخ دمشق - الجزء ( 2 ) - رقم الصفحة ( 283 / 306 ) الطبعة الثانية.
