العودة لصفحة البداية

العودة لفهرس الرواة

 

{ أَفَمَن كَانَ عَلَى بَيِّنَةٍ مِّن رَّبِّهِ وَيَتْلُوهُ شَاهِدٌ مِّنْهُ ( هود : 17 ) }

 

 عدد الروايات : ( 2 )

 

الشوكاني - فتح القدير الجامع بين فني الرواية والدراية - تفسير سورة هود : 9

تفسير قوله تعالى : { وَلَئِنْ أَذَقْنَا الْإِنسَانَ مِنَّا رَحْمَةً ثُمَّ نَزَعْنَاهَا مِنْهُ إِنَّهُ لَيَئُوسٌ كَفُورٌ }

 الجزء : ( 1 ) - رقم الصفحة : ( 651 )

 

[ النص طويل لذا استقطع منه موضع الشاهد ]

 

- .... وأخرج ابن أبي حاتم ، وابن مردويه ، وأبو نعيم في المعرفة ، عن علي بن أبي طالب ، قال : ما من رجل من قريش الا نزل فيه طائفة من القرآن ، فقال له رجل : ما نزل فيك ، قال : أما تقرأ سورة هود : { أَفَمَن كَانَ عَلَى بَيِّنَةٍ مِّن رَّبِّهِ وَيَتْلُوهُ شَاهِدٌ مِّنْهُ ( هود : 17 ) } رسول الله (ص) بينة من ربه ، وأنا : { شَاهِدٌ مِّنْهُ ( هود : 17 ) }.

 

- .... وأخرج ابن عساكر ، وابن مردويه من وجه آخر عنه ، قال : قال رسول الله (ص) : { أَفَمَن كَانَ عَلَى بَيِّنَةٍ مِّن رَّبِّهِ ( هود : 17 ) } أنا ، { وَيَتْلُوهُ شَاهِدٌ مِّنْهُ ( هود : 17 ) } علي.
