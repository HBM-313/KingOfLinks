العودة لفهرس الرواة

م

{ إِنَّمَا أَنتَ مُنذِرٌ وَلِكُلِّ قَوْمٍ هَادٍ ( الرعد : 7 ) }

 

 عدد الروايات : ( 2 )

 

القندوزي - ينابيع المودة لذوي القربى

 الجزء : ( 2 ) - رقم الصفحة : ( 73 )

 

34 -  أنا المنذر وعلي الهادي ، وبك يا علي يهتدي المهتدون من بعدي.

 

 

القندوزي - ينابيع المودة لذوي القربى

الجزء : ( 2 ) - رقم الصفحة : ( 246 )

 

693 - الحديث الخمسون ، عن ابن عباس (ر) ، قال : لما نزل قوله : { إِنَّمَا أَنتَ مُنذِرٌ وَلِكُلِّ قَوْمٍ هَادٍ ( الرعد : 7 ) } قال رسول الله (ص) أنا المنذر وعلي الهادي ، وبك يا علي يهتدي المهتدون ، رواه صاحب الفردوس.
