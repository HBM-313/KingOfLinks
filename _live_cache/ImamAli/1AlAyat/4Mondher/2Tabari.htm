العودة لفهرس الرواة

م

{ إِنَّمَا أَنتَ مُنذِرٌ وَلِكُلِّ قَوْمٍ هَادٍ ( الرعد : 7 ) }

 

 عدد الروايات : ( 2 )

 

الطبري - جامع البيان عن تأويل آي القرآن - سورة الرعد : 7  - القول في تأويل قوله تعالى

 { وَيَقُولُ الَّذِينَ كَفَرُوا لَوْلَا أُنزِلَ عَلَيْهِ آيَةٌ مِّن رَّبِّهِ إِنَّمَا أَنتَ مُنذِرٌ وَلِكُلِّ قَوْمٍ هَادٍ }

الجزء : ( 13 ) - رقم الصفحة : ( 442 )

 

- حدثني : الحسن ، قال : ثنا : محمد وهو ابن يزيد ، عن إسماعيل ، عن يحيى بن رافع ، في قوله : { إِنَّمَا أَنتَ مُنذِرٌ وَلِكُلِّ قَوْمٍ هَادٍ ( الرعد : 7 ) } قال : قائد ، وقال آخرون : هو علي بن أبي طالب (ر).

 

 

الطبري - جامع البيان عن تأويل آي القرآن - سورة الرعد : 7  - القول في تأويل قوله تعالى

 { وَيَقُولُ الَّذِينَ كَفَرُوا لَوْلَا أُنزِلَ عَلَيْهِ آيَةٌ مِّن رَّبِّهِ إِنَّمَا أَنتَ مُنذِرٌ وَلِكُلِّ قَوْمٍ هَادٍ }

 الجزء : ( 13 ) - رقم الصفحة : ( 442 )

 

- ذكر من قال ذلك : حدثنا : أحمد بن يحيى الصوفي ، قال : ثنا : الحسن بن الحسين الأنصاري ، قال : ثنا : معاذ بن مسلم ، ثنا : الهروي ، عن عطاء بن السائب ، عن سعيد ابن جبير ، عن ابن عباس ، قال : لما نزلت : { إِنَّمَا أَنتَ مُنذِرٌ ۖ وَلِكُلِّ قَوْمٍ هَادٍ ( الرعد : 7 ) } وضع (ص) يده على صدره ، فقال : أنا المنذر { وَلِكُلِّ قَوْمٍ هَادٍ ( الرعد : 7 ) } وأومأ بيده إلي منكب علي ، فقال : أنت الهادي يا علي ، بك يهتدي المهتدون بعدي ، وقال آخرون : معناه : لكل قوم داع.
