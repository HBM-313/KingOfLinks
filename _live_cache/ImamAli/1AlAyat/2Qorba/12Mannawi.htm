المناوي - فيض القدير شرح الجامع الصغير - حرف : الهمزة

الجزء : ( 1 ) - رقم الصفحة : ( 219 )

 

302 - أخلفوني بضم الهمزة واللام ، أي كونوا خلفائي في أهل بيتي علي وفاطمة وابنيهما وذريتهما ، فإحفظوا حقي فيهم وأحسنوا الخلافة عليهم باعظامهم واحترامهم ونصحهم والاحسان اليهم وتوقيرهم والتجاوز عن مسيئهم : { قُل لَّا أَسْأَلُكُمْ عَلَيْهِ أَجْرًا إِلَّا الْمَوَدَّةَ فِي الْقُرْبَى ( الشورى : 23 ) }.
