السيوطي

الجزء : ( 1 ) - رقم الصفحة : ( 182 )

[النص طويل لذا استقطع منه موضع الشاهد ]

- .... وقال غيره : كان بنو أميه يسبون علي بن أبي طالب (ع) في الخطبه ، فلما ولي عمر بن عبدالعزيز أبطله وكتب إلى نوابه بابطاله وقرأ مكانه : { إِنَّ اللَّهَ يَأْمُرُ بِالْعَدْلِ وَالْإِحْسَانِ ( النحل : 90 ) } فاستمرت قراءتها في الخطبة إلى الآن.
