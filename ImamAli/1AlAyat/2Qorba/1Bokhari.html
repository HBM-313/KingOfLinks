العودة لصفحة البداية

العودة لفهرس الرواة

{ قُل لَّا أَسْأَلُكُمْ عَلَيْهِ أَجْرًا إِلَّا الْمَوَدَّةَ فِي الْقُرْبَى ( الشورى : 23 ) }

ممم

عدد الروايات : (

2 )

صحيح البخاري - كتاب المناقب - باب : قول الله تعالى :

{ يَا أَيُّهَا النَّاسُ إِنَّا خَلَقْنَاكُم مِّن ذَكَرٍ وَأُنثَى وَجَعَلْنَاكُمْ شُعُوبًا وَقَبَائِلَ لِتَعَارَفُوا إِنَّ أَكْرَمَكُمْ عِندَ اللهِ أَتْقَاكُمْ ) ( الحجرات : 13 ) }

الجزء : ( 4 ) - رقم الصفحة : ( 154 )

3306 - باب ‏: حدثنا : ‏ ‏مسدد ‏ ، حدثنا : ‏ ‏يحيى ‏ ‏، عن ‏ ‏شعبة ‏ ‏، حدثني : ‏ ‏عبد الملك ‏ ‏، عن ‏ ‏طاوس ‏ ‏، عن ‏ ‏ابن عباس ‏‏(ر) ‏‏: { إِلَّا الْمَوَدَّةَ فِي الْقُرْبَى ( الشورى : 23 ) }قال : فقال سعيد ابن جبير ‏ : ‏قربى ‏ ‏محمد ‏ (ص) ‏، ‏فقال : ‏ ‏أن النبي (ص) ‏لم يكن بطن من ‏ ‏قريش ‏ ‏الا وله فيه قرابة ، فنزلت عليه : الا أن تصلوا قرابة بيني وبينكم.

صحيح البخاري - كتاب تفسير القرآن سورة حم عسق

باب : قوله : { إِلَّا الْمَوَدَّةَ فِي الْقُرْبَى }

الجزء : ( 6 ) - رقم الصفحة : ( 37 )

4541 - حدثنا : ‏ ‏محمد بن بشار ‏ ، حدثنا : ‏ ‏محمد بن جعفر ‏ ، حدثنا : ‏ ‏شعبة ‏ ‏، عن ‏ ‏عبد الملك بن ميسرة ،‏ ‏قال : سمعت ‏ ‏طاوسا ‏ ‏، عن ‏ ‏ابن عباس ‏ ‏(ر) ‏ ‏أنه سئل عن قوله ‏‏:‏ ‏{ إِلَّا الْمَوَدَّةَ فِي الْقُرْبَى ( الشورى : 23 ) }فقال سعيد ابن جبير ‏ : ‏قربى آل ‏ ‏محمد ‏ (ص) ‏، ‏فقال ابن عباس ‏ : ‏عجلت أن النبي ‏ (ص) ‏ ‏لم يكن بطن من ‏ ‏قريش ‏ ‏الا كان له فيهم قرابة ، فقال : ‏ ‏الا أن تصلوا ما بيني وبينكم من القرابة.
