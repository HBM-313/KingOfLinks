العودة لصفحة البداية

العودة لفهرس الرواة

{ قُل لَّا أَسْأَلُكُمْ عَلَيْهِ أَجْرًا إِلَّا الْمَوَدَّةَ فِي الْقُرْبَى ( الشورى : 23 ) }

مم

عدد الروايات : (

1 )

القرطبي - تفسير القرطبي = الجامع لأحكام القرآن - سورة الشورى : 23

قوله تعال

ى : ( ذَلِكَ الَّذِي يُبَشِّرُ اللهُ عِبَادَهُ الَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَاتِ )

الجزء : ( 16 ) - رقم الصفحة : ( 21 )

[النص طويل لذا استقطع منه موضع الشاهد ]

- .... قوله تعالى : { قُل لَّا أَسْأَلُكُمْ عَلَيْهِ أَجْرًا إِلَّا الْمَوَدَّةَ فِي الْقُرْبَى( الشورى : 23 ) }فيه مسألتان : الأولى قوله تعالى : { قُل لَّا أَسْأَلُكُمْ عَلَيْهِ أَجْرًا ( الشورى : 23 ) }أي قل يا محمد لا أسئلكم على تبليغ الرسالة جعلا : { إِلَّا الْمَوَدَّةَ فِي الْقُرْبَى ( الشورى : 23 ) }قال الزجاج : الا المودة ، استثناء ليس من الأول ، أي الا أن تودوني لقرابتي فتحفظوني ، والخطاب لقريش خاصة.

- .... وفي البخاري ، عن طاوس ، عن ابن عباس : أنه سئل عن قوله تعالى : { إِلَّا الْمَوَدَّةَ فِي الْقُرْبَى ( الشورى : 23 ) }فقال سعيد ابن جبير : قربى آل محمد.

- .... وقيل : القربى قرابة الرسول (ص) ، أي لا أسئلكم أجرا إلا أن تودوا قرابتي وأهل بيتي ، كما أمر باعظامهم ذوي القربى.

- .... وفي رواية سعيد ابن جبير ، عن ابن عباس : لما أنزل الله عز وجل : { قُل لَّا أَسْأَلُكُمْ عَلَيْهِ أَجْرًا إِلَّا الْمَوَدَّةَ فِي الْقُرْبَى( الشورى : 23 ) }قالوا : يا رسول الله ، من هؤلاء الذين نودهم، قال : علي وفاطمة وأبناؤهما.
