العودة لصفحة البداية

العودة لفهرس الرواة

{ أَفَمَن كَانَ عَلَىبَيِّنَةٍ مِّن رَّبِّهِ وَيَتْلُوهُ شَاهِدٌ مِّنْهُ

( هود: 17 ) }

عدد الروايات : (

3 )

السيوطي - الدر المنثور في التفسير بالمأثور - هود : 17

الجزء : ( 4 ) - رقم الصفحة : ( 409 / 410 )

[النص طويل لذا استقطع منه موضع الشاهد ]

- .... أخرج ابن أبي حاتم ، وابن مردويه ، وأبو نعيم في المعرفة ، عن علي بن أبي طالب (ر) ، قال :ما من رجل من قريش الا نزل فيه طائفة من القرآن ، فقال له رجل : ما نزل فيك ، قال :أما تقرأ سورة هود : { أَفَمَن كَانَ عَلَىبَيِّنَةٍ مِّن رَّبِّهِ وَيَتْلُوهُ شَاهِدٌ مِّنْهُ ( هود: 17 ) }رسول الله (ص) : { عَلَىبَيِّنَةٍ مِّن رَّبِّهِ ( هود: 17 ) }وأنا : { شَاهِدٌ مِّنْهُ ( هود: 17 ) }.

- .... وأخرج ابن مردويه ، وابن عساكر ، عن علي (ر) في الآية ، قال رسول الله : { عَلَىبَيِّنَةٍ مِّن رَّبِّهِ ( هود: 17 ) }وأنا : { شَاهِدٌ مِّنْهُ ( هود: 17 ) }.

- .... وأخرج ابن مردويه من وجه آخر ، عن علي (ر) ، قال : قال رسول الله (ص) : { أَفَمَن كَانَ عَلَىبَيِّنَةٍ مِّن رَّبِّهِ ( هود: 17 ) }أنا ، { وَيَتْلُوهُ شَاهِدٌ مِّنْهُ ( هود: 17 ) }قال : علي.
