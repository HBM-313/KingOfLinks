الطبري

- تفسير الطبري = جامع البيان عن تأويل آي القرآن - سورة البينة - وقوله :

{ إِنَّ الَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَاتِ أُولَئِكَ هُمْ خَيْرُ الْبَرِيَّةِ ( البينة : 7 ) }

الجزء : ( 24 ) - رقم الصفحة : ( 556 )

- حدثنا : ابن حميد ، قال : ثنا : عيسى بن فرقد ، عن أبي الجارود ، عن محمد بن علي : { إِنَّ الَّذِينَ آمَنُوا وَعَمِلُوا الصَّالِحَاتِ أُولَئِكَ هُمْ خَيْرُ الْبَرِيَّةِ ( البينة : 7 ) } فقال النبي (ص) : أنت يا علي وشيعتك.
