العودة لفهرس الرواة

{ وَتَعِيَهَا أُذُنٌ وَاعِيَةٌ ( الحاقة : 12 ) }

عدد الروايات : (

2 )

المتقي الهندي - كنز العمال في سنن الأقوال والأفعال

الجزء : ( 13 ) - رقم الصفحة : ( 177 )

36525 - عن علي ، قال : قال رسول الله (ص) يا علي إن الله أمرني أن أدنيك وأعلمك لتعي ، وأنزلت هذه الآية : { وَتَعِيَهَا أُذُنٌ وَاعِيَةٌ ( الحاقة : 12 ) } فأنت أذن واعية لعلمي.

36526 - عن علي في قوله : { وَتَعِيَهَا أُذُنٌ وَاعِيَةٌ ( الحاقة : 12 ) } قال : قال لي رسول الله (ص) : سألت الله أن يجعلها أذنك يا علي فما سمعت من رسول الله (ص) شيئا فنسيته.
