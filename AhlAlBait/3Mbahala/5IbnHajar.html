العودة لفهرس الرواة

{ فَقُلْ تَعَالَوْا نَدْعُ أَبْنَاءَنَا وَأَبْنَاءَكُمْ وَنِسَاءَنَا وَنِسَاءَكُمْ وَأَنفُسَنَا وَأَنفُسَكُمْ

( آل عمران : 61 ) }

عدد الروايات : (

2 )

ابن حجر العسقلاني - فتح الباري شرح صحيح البخاري - كتاب فضائل الصحابة

باب : مناقب علي بن أبي طالب القرشي الهاشمي أبي الحسن (ر)

الجزء : ( 7 ) - رقم الصفحة : ( 60 )

[النص طويل لذا استقطع منه موضع الشاهد ]

- .... وقوله : لما نزلت : { فَقُلْ تَعَالَوْا نَدْعُ أَبْنَاءَنَا وَأَبْنَاءَكُمْ ( آل عمران : 61 ) } دعا عليا وفاطمة والحسن والحسين عليهم السلام ، فقال : اللهم هؤلاء أهلي.

ابن حجر العسقلاني - الإصابة في تمييز الصحابة - تتمة حرف : العين المهملة

تتمة القسم الأول من حرف : العين - 5704 - علي بن أبي طالب الهاشمي (ر)

الجزء : ( 4 ) - رقم الصفحة : ( 468 )

[النص طويل لذا استقطع منه موضع الشاهد ]

- .... وأنزلت هذه الآية :{ فَقُلْ تَعَالَوْا نَدْعُ أَبْنَاءَنَا وَأَبْنَاءَكُمْ وَنِسَاءَنَا وَنِسَاءَكُمْ وَأَنفُسَنَا وَأَنفُسَكُمْ ( آل عمران : 61 ) } فدعا رسول الله (ص) عليا وفاطمة وحسنا وحسينا (ع) ، فقال : اللهم هؤلاء أهلي.
